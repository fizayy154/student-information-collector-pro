A beginner-level Python project to collect student information and calculate percentages.
# Student Information Collector

This is a beginner-level Python program that collects student information and displays a complete student profile.

#Features
- Takes student name, father name, age, gender, and city as input
- Takes matric and FSc marks as input
- Calculates matric percentage
- Calculates FSc percentage
- Calculates average percentage
- Shows scholarship eligibility status

# Purpose
This project is made for basic Python practice, including input, variables, calculations, if-else statements, and print output.

## Language Used
Python
